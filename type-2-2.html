2. Based on Refractive Index Profile

Step Index Fiber

  • Abrupt change in refractive index
  • Light reflects in zig-zag paths

Types:

  • Single-mode step index
  • Multimode step index

Used in: Simple, low-cost systems

Graded Index Fiber

  • Refractive index gradually decreases
  • Light follows curved path
  • Reduces signal distortion

Used in: High-speed LAN
